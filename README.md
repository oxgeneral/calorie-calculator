# Научный калькулятор калорий для снижения веса

Интерактивный калькулятор, основанный на современных научных исследованиях в области диетологии и метаболизма, который помогает составить оптимальный план питания для эффективного и здорового снижения веса.

## Ключевые функции

- **Персонализированные рекомендации** на основе пола, возраста, веса, роста и уровня активности
- **Поддержка разных типов диет**: 
  - Стандартный научный подход
  - Циклическое углеводное питание (5+2)
  - Углеводная загрузка вечером
  - Интервальное голодание (16:8)
- **Детальный расчет макронутриентов**: оптимальное соотношение белков, жиров и углеводов
- **Прогноз снижения веса** с учетом научных данных и метаболических адаптаций
- **Рекомендации по тренировкам** и образу жизни
- **Визуализация данных** с помощью интерактивных графиков

## Технологии

- Next.js
- React
- TypeScript
- Tailwind CSS
- Recharts для визуализации данных

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/username/calorie-calculator.git
cd calorie-calculator

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

# Собрать проект для продакшн
npm run build

# Запустить собранный проект
npm start
```

## Деплой на Vercel

Проект оптимизирован для деплоя на платформе Vercel. Для размещения:

1. Создайте аккаунт на [Vercel](https://vercel.com)
2. Подключите репозиторий GitHub с проектом
3. Настройте и опубликуйте проект
